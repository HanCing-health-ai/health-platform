# 風險分流規則 V1
版本：1.0 | 建立日期：2026-03-15
審閱人：HanCing（待專業顧問確認）

## 系統定位聲明
本規則用於行為模式導向的健康管理工具，所有輸出為行為調整建議，不涉及醫療診斷或治療。

## A 類 — 一般調理型（繼續全流程）
條件（以下皆符合才判定為 A）：
- 無急性疼痛描述
- 無 C 類觸發關鍵詞
- 不適部位為肌肉、筋膜、關節一般緊繃或痠痛
- 狀況持續時間為數天至數週的慢性累積型

處理：正常進入分析流程

## B 類 — 恢復期加強監控型（繼續流程 + 師傅特別提醒）
條件（符合任一即判定為 B）：
- 近期曾接受手術或醫療處置（3 個月內）
- 有舊傷史且本次描述與舊傷部位重疊
- 年齡 ≥65 且描述關節不適
- 孕期中
- 長期服用抗凝血藥物（如阿斯匹靈）
- 心臟支架術後

處理：進入分析流程，但輸出加入「建議師傅特別評估」標記，師傅端收到額外警示通知

## C 類 — 建議就醫（停止流程）
觸發任一關鍵詞即停止，回覆就醫建議。

### C 類觸發關鍵詞庫 V1

急性症狀類：
- 痛到無法呼吸
- 胸口悶痛、胸痛
- 突然暈眩、突然站不穩
- 手腳麻痺（突然發作）
- 臉部歪斜、說話困難
- 視線模糊（突然）
- 左手臂痠麻合併胸悶
- 冷汗合併頭暈

嚴重外傷類：
- 骨折
- 脫臼
- 撞傷後無法移動

神經症狀類：
- 大小便失禁、控制困難
- 單側手腳突然無力
- 吞嚥困難

腫脹發炎類：
- 關節紅腫發熱（急性）
- 不明原因持續腫脹超過一週

疾病相關類：
- 發燒超過 38 度
- 癌症、腫瘤部位
- 術後傷口未癒合
- 骨質疏鬆史合併跌倒後疼痛
- 糖尿病合併足部傷口未癒

處理：
1. 停止分析流程
2. 回覆標準文字：「您描述的狀況建議優先諮詢醫療專業人員，調理服務適合在醫療評估後進行。」
3. 記錄觸發關鍵詞（後台統計用，不顯示給使用者）
