# compliance-check
description: 掃描 ConditionAI 輸出內容，確認無醫療禁止詞彙。適用於所有涉及 client_output 的程式碼審查。

## 禁止詞彙清單
診斷、確診、病因、病症、症狀、疾病、病變、治療、療效、醫治、
處方、藥物、痊癒、康復、治癒、風險、併發症、惡化、緊急、危險

## 審查範圍
- FastAPI 回傳的 client_output 所有欄位
- 前端顯示 AI 建議的所有元件
- System Prompt 中的 client 相關段落

## 審查方式
逐欄位掃描，發現違規立即回報：檔案路徑 + 行號 + 違規詞彙

## 合規語言範例
❌ 不合規：「建議治療肩頸疼痛」
✅ 合規：「建議調整肩頸使用習慣」
❌ 不合規：「診斷為久坐症狀」
✅ 合規：「觀察到久坐行為模式」
