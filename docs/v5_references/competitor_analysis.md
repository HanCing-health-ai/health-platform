# ConditionAI

## 競品分析報告
健康調理軟體市場　競爭態勢與差異化定位

| 報告日期 | 2026 年 3 月 |
| -------- | ------------ |
| 分析對象 | 健康調理 SaaS 及 AI Wellness 工具市場 |
| 地理範圍 | 台灣市場為主，兼顧亞太與全球佈局 |
| 報告目的 | 評估現有競品威脅程度，確認 ConditionAI 差異化定位 |
| 版本 | V1.0 - 內部策略文件 |

# 一　市場概況與競品地圖
台灣市場特殊性：傳統整復推拿產業有 4,800 家業者、10 萬名從業人員，是亞洲傳統調理密度最高的市場之一，但尚無任何本土數位化工具專門服務此族群。

- 第一級（直接）：Noterro、Zenoti、Mindbody
- 第二級（間接）：ClinicSense、MassageBook、Jotform AI
- 第三級（環境）：台灣 × Google AI 健康、本土新創

# 二　第一級威脅：直接競品深度分析
- **Noterro (中高威脅)**：AI Scribe 已率先上線。但弱點是「只記錄，不分析」，缺乏行為模式分析與縱向追蹤。
- **Zenoti (中威脅)**：資金最龐大，但月費過高 (約 NT$12,000)，直接排除台灣 95% 小型工作室。
- **Mindbody (低-中威脅)**：消費者市集平台媒合模式，但目前不在台灣市場。

# 三　第二級與第三級威脅
包含單純預約系統 ClinicSense 以及政府醫療 AI 佈局，均屬行政工具或純醫療範圍，不直接威脅日常民俗調理市場。

# 四　ConditionAI 差異化總結與矩陣
ConditionAI 在「行為模式分析」「雙向洞察輸出」「可解釋式建議」「縱向回訪」「風險合規」五個維度上，目前市場上無任何競品達到相同功能深度。這五個維度構成核心護城河。定價設計也完美符合台灣中小型從業者的承受範圍（NT$1,500 級距）。

# 五　策略建議與行動優先順序
**立即行動（0-3 個月）**：加快試點，建立社群壁壘。明確 V2 AI Scribe 開發。
**最高優先風險提醒**：Noterro 的 AI Scribe 已上線，必須把握視窗期，利用「分析與縱向資料」甩開語音記錄的工具定位。
